# Failure Modes Brief — How Blockchains Fail (Draft)

## Technical failures
- **51% attacks:** majority hash/stake reorgs chain
- **Consensus bugs:** software divergence creates forks
- **Network partitions:** chain splits from connectivity issues

## Economic failures
- **Incentive misalignment:** rewards don’t match honest behavior
- **High fees / congestion:** system unusable for small users

## Governance failures
- **Centralization creep:** validators/miners concentrate
- **Capture:** a small group controls updates or policy

## Operational failures
- **Key loss / theft:** irreversible loss of funds
- **Smart contract bugs:** irreversible logic errors
