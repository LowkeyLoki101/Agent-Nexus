# RLM Local API

## Install
```
pip install -r requirements.txt
```

## Run
```
uvicorn rlm_api:app --host 0.0.0.0 --port 8089
```

## Auth
- Set API token in `rlm_api.py` (API_TOKEN)
- Send header: `X-Token: <token>`

## Example
```
curl -X POST http://localhost:8089/summarize \
  -H "Content-Type: application/json" \
  -H "X-Token: CHANGE_ME" \
  -d '{"text":"Hello world","mode":"public"}'
```
