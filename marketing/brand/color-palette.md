## Gradients

| Name | CSS | Usage |
|------|-----|-------|
| Gold Shimmer | `linear-gradient(135deg, #E5A824, #F0C850)` | Premium CTAs, hero elements |
| Dark Depth | `linear-gradient(180deg, #1A1A2E, #16213E)` | Page backgrounds |

## Typography

- **Headings**: Inter or system sans-serif, bold weight
- **Body**: Inter or system sans-serif, regular weight
- **Monospace**: JetBrains Mono or system monospace (for code/technical content)

## Logo Usage

- Primary logo uses Gold (#E5A824) on Dark Charcoal (#1A1A2E)
- Reversed logo uses Dark Charcoal on light backgrounds
- Minimum clear space: 1x the height of the "C" in CB
- Company name format: CB | CREATIVES (always uppercase, pipe separator)
